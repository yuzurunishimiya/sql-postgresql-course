'CREATE TABLE' : Keyword
'cities' : Identifier

What are they?
    - Keyword: Tell the database that we want to do something. Always written out in capital letters.
    - Identifier: Tell the database what thing we want to act on. Always written out in lower case lettes.


Column data types
- 'VARCHAR(50)': Variable length character. Text! if we put in a string longer than chars, we'll get an error
- INTEGER: Number without a decimal. -2,147,483,647 to 2,147,483,647. Anyting larger or smaller = error
